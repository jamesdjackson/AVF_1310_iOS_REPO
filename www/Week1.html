2. Visit Programmableweb.com and search for JSON-P data sources. Why would we need JSON-P, specifically?

JSON-P is essentially a work around to allow an application to retrieve data form an external domain. This would ordinarily be treated as a cross domain violation and not be allowed. However, the request is sent as JavaScript and handled in such a way that the data is allowed to be retrieved or interpreted as script, rather than parsed as a JSON data set.
